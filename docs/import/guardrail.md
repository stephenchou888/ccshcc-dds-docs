👋 主线（**低等级道路或者高速，同时支持国际项目中的DIY式原则转换布设**）：

- :penguin: 如果数据齐全，包括桥表、隧道表、填挖方数据都比较完整，可以采用全自动化的模式进行，参考主线自动设计模式。

- :penguin: 如果数据不齐全，甚至只有一张总体图，则采用半自动模式进行设计。

- :penguin: 如果只有部分数据，比如桥隧表，可以先按照全自动模式进行设计，并将护栏绘制到总体图上，然后用编辑工具进行修正。

- :penguin: 综上所述，所谓B模式仅仅是加入了互通人工干预布设的A模式加强版。

- **A、B模式最终版会进行功能合并形成全新版本护栏自动化布设工具集。**
